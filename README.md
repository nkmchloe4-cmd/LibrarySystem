# Library System
Detta projekt är en konsolapplikation som simulerar ett enkelt bibliotekssystem.
Uppgiften fokuserar på objektorienterad programmering i C#.


# Projektet demonstrerar:
- Klasser och objekt  
- Inkapsling  
- Komposition  
- Interface och polymorfism  
- Grundläggande algoritmer  
- Enhetstester med xUnit  

# Funktionalitet  
✔ Hantera böcker  
- Lägga till böcker  
- Visa alla böcker  
- Söka på titel, författare eller ISBN  
- Sortera böcker (titel eller år)  


✔ Hantera medlemmar  
- Lägga till medlemmar  
- Visa medlemmar  
- Spåra aktiva lån  


✔ Utlåning  
- Låna bok  
- Returnera bok
- Kontrollera försenade lån

✔ Statistik  
- Totalt antal böcker  
- Antal utlånade böcker
- Mest aktiv låntagare

# Enhetstester (xUnit)  
Testprojektet innehåller 10 tester:
- Book: 3 tester  
- Loan: 3 tester  
- Search (ISearchable): 2 tester  
- Statistics/algoritmer: 2 tester  

# Kör projektet  
I Visual Studio:
- Markera **LibrarySystem** som startup‑projekt  
- Tryck **F5**  

# Kör tester  
Via Visual Studio:
- Öppna **Test Explorer**  
- Klicka **Run All Tests**
